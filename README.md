
# 📊 Amazon Sales Analysis Dashboard (Excel)

## 📌 Project Overview

This project analyzes Amazon sales data using Microsoft Excel to uncover sales trends, customer behavior, regional performance, and delivery efficiency.

The dashboard was built using Pivot Tables, Pivot Charts, XLOOKUP, Conditional Formatting, and Slicers.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- Slicers
- Conditional Formatting

## 📊 Key KPIs

- Total Revenue
- Average Order Value
- Highest Selling Product Category
- Top Performing Regions
- Effective Sales
- Cancellation Rate
- Fast Delivery Percentage

---

## 🔍 Key Insights

- Electronics generated the highest revenue.
- North region recorded the highest cancellation rate.
- Around 30% of orders were cancelled.
- Average order value was approximately ₹352.
- Fast delivery rate was 11%.

---

## 💡 Recommendations

- Improve logistics in high-cancellation regions.
- Increase fast delivery capabilities.
- Maintain inventory for high-demand products.
- Monitor regional performance regularly.
